# SPNSS Website

This is the SvelteKit-based website for SPNSS EOOD.

## Development

```bash
npm install
npm run dev
```

## Testing

```bash
npm run test:e2e
```

## Deployment

The site is automatically deployed to GitHub Pages via GitHub Actions.
Previews are generated for each Pull Request.
